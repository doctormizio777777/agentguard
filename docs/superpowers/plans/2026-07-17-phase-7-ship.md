# Phase 7 Ship Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a clean one-command Docker demo and a factual final README for AgentGuard.

**Architecture:** Reuse the existing backend schema, service layer, and deterministic dashboard seed. A backend entrypoint performs schema initialization and guarded seeding before Uvicorn; a multi-stage frontend image serves the existing dashboard against the backend service exposed through the browser-facing host URL.

**Tech Stack:** FastAPI, Python 3.11, SQLite, Uvicorn, Next.js 15, Node 22, pnpm, Docker Compose.

## Global Constraints

- The seeded demo must not require an OpenAI/OpenRouter key.
- Seed only when the SQLite ledger is empty; never reset or delete existing data.
- Do not change policy, intent firewall, ledger, MCP, or dashboard behavior beyond packaging/README needs.
- No secrets in tracked files; `.env` remains ignored and `.env.example` contains placeholders only.
- Backend and frontend remain separate services on ports 8000 and 3000.
